# EM-FilterSets（EMフィルターセット）

MV 制作やシナリオ重視の動画向けの映像エフェクト集 YMM4（ゆっくりMovieMaker4）プラグインです。
AfterEffects 等で使われる「光・質感・色調・動き・歪み」系の演出を、GPU（HLSL カスタムシェーダー / Direct2D）で多数まとめて提供します。

> **EM** は **Emoi** の略です。はい。

---

## 特徴

- 現時点で**17 種類の映像エフェクト**を1プラグインに同梱
- すべて **GPU 処理**（HLSL カスタムシェーダー／Direct2D 組み込みエフェクト）で軽快
- エフェクト追加メニューのカテゴリ **「EMフィルターセット」** にまとめて表示
- 各エフェクト名は **`em_`** 接頭辞付き（標準エフェクトとの名前衝突を回避）
- 主要パラメータは **中間点やキーフレームアニメーション対応**

---

## 動作環境

- ゆっくりMovieMaker4（YMM4 v4.35 以降を推奨）
- .NET 10 ランタイム（YMM4 本体に準拠）

## インストール

1. [Releases](../../releases) から `EM-FilterSets.v.x.x.x.ymme` をダウンロード
2. ファイルを YMM4 のウィンドウにドラッグ＆ドロップ（または YMM4 のプラグイン管理から追加）
3. YMM4 を再起動

> 手動で配置する場合は、`EM-FilterSets` フォルダごと
> `（YMM4フォルダ）\user\plugin\` 以下に置いてください。

## 使い方

アイテム（画像・動画・立ち絵など）の「映像エフェクト」追加メニュー →
カテゴリ **「EMフィルターセット」** から `em_◯◯` を選択して追加します。

## 収録エフェクト一覧（17種）

### 光・グロー系
| 名称 | 内容 | 主なパラメータ |
|---|---|---|
| em_ディフュージョン | 全体を柔らかく拡散発光（ソフトフォーカス） | ぼかし量・強さ・合成モード |
| em_ハレーション | ハイライトから漏れる色付きのにじみ（フィルム調） | しきい値・にじみ量・強さ・色・合成モード |
| em_レンズフレア | 手続き生成のコア／ゴースト／ハロ | 光源X/Y・強さ・大きさ・合成モード |
| em_シェイプフレア | YMM4標準図形をゴーストにしたフレア | 図形・光源X/Y・ゴースト数・広がり・大きさ・強さ・色・合成モード |
| em_リムライト | アルファ縁の光源側を光らせる（光の回り込み） | 光源X/Y・縁幅・ぼかし・強さ・色・合成モード |
| em_ストライプ光 | 流れる光の帯（直線／同心円／放射状） | モード・角度/中心・間隔/本数・帯幅・速度・強さ・合成モード |
| em_クロマティックアベレーション | 中心からの距離に応じたRGBの色ずれ（色収差） | 強さ・中心X/Y |

### 質感・劣化系
| 名称 | 内容 | 主なパラメータ |
|---|---|---|
| em_フィルムグレイン | フィルム粒子ノイズ（時間変化・FPS非依存） | 強さ・粒の大きさ・モノクロ切替 |
| em_スキャンライン | 走査線（CRT風・スクロール対応） | 本数・強さ・スクロール速度 |
| em_ビネット | 周辺減光 | 強さ・半径・ぼかし |

### 色調・トーン系
| 名称 | 内容 | 主なパラメータ |
|---|---|---|
| em_ポスタリゼーション | 階調を落とす（減色） | 階調数 |

### 動き系
| 名称 | 内容 | 主なパラメータ |
|---|---|---|
| em_ジッター | 微振動・小刻みなブレ（FPS非依存） | 振幅・間隔(秒) |
| em_ストロボ | フラッシュ的な明滅（FPS非依存） | 速さ・強さ・鋭さ |

### ディストーション系
| 名称 | 内容 | 主なパラメータ |
|---|---|---|
| em_波形歪み | 水面のような揺らぎ | 揺れ幅・波長・速度・方向 |
| em_ピクセレート | モザイク（正方形ブロック） | ピクセルサイズ |
| em_カレイドスコープ | 万華鏡（鏡の折り返し反復） | 分割数・回転・中心X/Y |
| em_フィードバック | 反復ズーム（トンネル／インフィニティミラー） | ズーム率・回転・減衰・繰り返し数・中心X/Y |

> **合成モード対応**のエフェクト（ディフュージョン／ハレーション／レンズフレア／シェイプフレア／リムライト／ストライプ光）は、
> YMM4 標準の通常・乗算・スクリーン・オーバーレイ・加算 などから合成方法を選べます。

## ビルド方法（開発者向け）

- 環境変数または `Directory.Build.props` の `YMM4DirPath` に「開発用 YMM4」フォルダのパスを設定
- HLSL のコンパイルに **Windows SDK の `fxc.exe`** が必要（ビルド時に `.hlsl` → `.cso` へコンパイルし埋め込み）

```sh
# 開発用 YMM4 のプラグインフォルダへ配置（デバッグ用）
dotnet build -c Release -p:Platform=x64

# 配布用 .ymme を生成（publish\EM-FilterSets.v.<Version>.ymme）
dotnet build -c Release -p:Platform=x64 -p:PackYmme=true

# C# のみ検証（fxc 不要・シェーダーコンパイルをスキップ）
dotnet build -p:SkipHlsl=true -p:SkipPluginCopy=true
```
